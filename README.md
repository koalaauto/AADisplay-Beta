# AADisplay-Beta

> ⚠️ **Archived & unmaintained.** A personal fork of [`Nitsuya/AADisplay`](https://github.com/Nitsuya/AADisplay) for experimenting with mirroring apps onto the Android Auto screen via a `VirtualDisplay`. No longer developed — kept here for reference only. Still licensed under GPL-3.0, same as upstream.

## Where the project went

**Want the open-source module?** Head upstream to [`Nitsuya/AADisplay`](https://github.com/Nitsuya/AADisplay) — the original open-source project for the LSPosed / Xposed approach.

**Want a maintained, ready-to-use app?** We took what we learned here and *independently* built **KoalaMirror** — an actively-developed product for casting Android apps onto the Android Auto screen, with regular updates and proper support. No Xposed-module tinkering required.

👉 **[koalamirror.com](https://koalamirror.com)**
